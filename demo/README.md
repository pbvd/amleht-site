# amleht | EU — Duco Demo Package

Complete demo materials for the amleht × Duna partnership presentation.

## Interactive Demos

Open the `.html` files directly in Chrome — no server or dependencies needed (requires internet for React CDN).

| File | Description |
|------|-------------|
| `portal-onboarding.html` | Duna B.V. sovereign onboarding journey — 6 phases from Welcome to Live with blockchain-anchored certificate. Clickable progress bar, auto-play demo mode. |
| `fictief-nv-dashboard.html` | Fictief NV compliance dashboard — Overview with 10-agent status, Chat with streaming agent responses, Compliance Ledger showing blockchain-anchored audit trail from genesis certificate through all assessments. |
| `nebius-inference-proof.html` | EU-sovereign inference proof — terminal-style demo showing agent assessments running on Nebius Token Factory in Helsinki, Finland. Token counts, latency, zero data retention. |
| `unified-trust-bundle.html` | Partnership vision — shows Duna identity chain and amleht infrastructure chain converging into a unified compliance passport. Click "Show Convergence" for animation. |
